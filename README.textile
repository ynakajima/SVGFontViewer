h1. SVGFontViewer

h2. Overview

SVGFontViewerはSVGフォントを閲覧するためのウェブアプリケーションです。
含まれているグリフの一覧と詳細を閲覧できます。

SVGFontViewer is a tool for SVG font to see in a browser.
You can preview the list of SVG fonts and glyphs details.
